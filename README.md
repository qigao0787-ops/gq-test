## 复习建议

### 时间安排

| 阶段 | 时间 | 内容 |
|------|------|------|
| 第一遍 | 3天 | 过所有 normal 版，建立知识骨架 |
| 第二遍 | 5天 | 读 strengthen 增强版，重点理解原理 |
| 第三遍 | 2天 | 只看高频问题表格，模拟口述回答 |
| 面试前 | 30分钟 | 看 normal 版的面试高频表格速记 |

### 面试节奏

```
1. 自我介绍 (1分钟): 姓名+年限+技术栈+亮点项目一句话
2. 项目深挖 (15分钟): STAR法 + 准备2-3个亮点
3. 八股文 (20分钟): 集合/并发/JVM/MySQL/Redis
4. 系统设计 (10分钟): SNAKE框架回答
5. 反问环节: 团队技术栈/业务方向/成长路径
```
