# 面试语音辅助工具 (隐蔽安全版)

> **核心原则：电脑上没有任何可见窗口，答案只在手机上显示。**
> 面试官共享屏幕时 100% 看不到。

---

## 工作原理

```
你的电脑 (共享屏幕中)              你的手机 (面试官看不到)
┌───────────────────────┐          ┌─────────────────┐
│                       │          │                 │
│  后台进程 (无窗口)     │  WiFi    │  浏览器显示:    │
│  ├─ 麦克风录音        │─────────→│  ├─ 识别的问题  │
│  ├─ Whisper语音识别   │ WebSocket│  ├─ 匹配的答案  │
│  ├─ 知识库搜索        │          │  └─ 手动搜索框  │
│  └─ Web服务(端口9900) │          │                 │
│                       │          │  (像看时间一样   │
│  ★ 屏幕上什么都没有   │          │   偶尔瞟一眼)   │
│                       │          │                 │
└───────────────────────┘          └─────────────────┘
```

## 快速开始

### 1. 安装

```bash
cd interview-assistant
pip install -r requirements.txt
```

### 2. 使用方式

| 模式 | 命令 | 需要 | 说明 |
|------|------|------|------|
| **完整模式** ⭐ | `python app.py` | 麦克风+Whisper | 自动识别+推送到手机 |
| **纯搜索Web** | `python search_only.py --web` | 无 | 手机手动输入搜索 |
| **终端搜索** | `python search_only.py` | 无 | 终端输入(测试用) |

### 3. 面试时操作

```bash
# 面试前5分钟执行:
python app.py

# 终端会显示:
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
#   ✅ 手机浏览器打开: http://192.168.1.100:9900
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

# 手机连同一WiFi，浏览器打开上面的地址
# 然后最小化终端，开始面试
```

## 安全保障

| 场景 | 安全性 | 原因 |
|------|--------|------|
| 面试官要求共享桌面 | ✅ 安全 | 电脑上没有任何可见窗口 |
| 面试官要求共享全屏 | ✅ 安全 | 后台进程不会显示在屏幕上 |
| 面试官查看任务栏 | ✅ 安全 | 终端可以最小化,看起来就是普通命令行 |
| 面试官要求关闭所有窗口 | ✅ 安全 | 关闭终端=关闭程序,手机端断开 |
| 录屏 | ✅ 安全 | 录的是你的屏幕,手机不在录制范围 |

### 额外安全建议

1. **终端伪装**: 运行后最小化终端,或用系统分屏把它藏在不显眼的位置
2. **手机放置**: 放在笔记本电脑下方/键盘旁,低头时像是在看笔记
3. **网页标题**: 页面标题是"备忘录",万一被看到也不突兀
4. **断网安全**: 完全离线运行,不会有网络请求被监控到

## 配置

编辑 `config.py`:

| 配置 | 默认 | 说明 |
|------|------|------|
| `WEB_PORT` | 9900 | Web服务端口 |
| `WHISPER_MODEL` | base | 模型大小(tiny更快/small更准) |
| `SILENCE_DURATION` | 1.5 | 静音多少秒算一句话说完 |
| `TOP_K_RESULTS` | 3 | 返回几条结果 |

## 常见问题

**Q: 手机打不开网页？**
- 确认手机和电脑连同一个WiFi
- 检查电脑防火墙是否拦截了端口9900
- Windows: 控制面板→防火墙→允许入站9900端口
- Mac: 一般不需要额外配置

**Q: 语音识别不准？**
- 改 `WHISPER_MODEL = "small"` (更准但慢2倍)
- 或直接用手机端手动输入搜索

**Q: 没有麦克风/不想用语音？**
- 用纯搜索模式: `python search_only.py --web`
- 手机端手动输入关键词搜索

**Q: 面试用的是耳机,麦克风收不到面试官声音？**
- 方案1: 把系统音频输出也录进来(需要虚拟声卡如 BlackHole/VB-Cable)
- 方案2: 用手机端手动输入关键词(最稳妥)
